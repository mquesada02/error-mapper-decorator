# Security Policy

## Supported versions

The latest published version on npm receives security fixes.

## Reporting a vulnerability

Please report security issues **privately**, not via public issues:

- Preferred: GitHub → the repository's **Security** tab → **Report a vulnerability**
  (private advisory).
- Or email **mquesada@itba.edu.ar**.

You can expect an initial response within a few days. Once a fix is released,
the advisory will be published with credit (if you'd like).
